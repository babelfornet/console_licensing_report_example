# Babel Licensing Report Example

This console application demonstrates how to use Babel Licensing Report functionality including:
- License validation with usage tracking
- Floating license request and release
- Custom license restrictions (MemoryRestriction)
- License report sending to the Babel Licensing Server

## Requirements

- .NET 9.0 or higher
- Babel.Licensing package (v11.5.0)

## Usage

Run the application and select from the menu options:
1. Validate License (simulate usage)
2. Send License Report
3. Exit
